Conhecendo por cima um pouco sobre as Tag's h1 até h6:

As Tags que usamos para criarmos títulos em paginas web são: h1, h2, h3, h4, h5, h6.

  1. Tag h1 - Representa o título de uma página web, h1 tem a importância mais alta.
  2. Tag h2 - Já a h2 são muito usadas para subtítulos.
  3. Tag h3 - Muito utilizadas para trechos de destaque.
  4. Tag h4, h5, h6 - Estas já são tags que raramentes é necessario utilizar.

      5. "Tag h4 -Sendo utilizada na pratica nesta demonstração, em um nivel de importância tem bem menos utilização."

      "Tag h5 -Sendo utilizada na pratica nesta demonstração, em um nivel de importância tem bem menos utilização e como podem ver ela tem uma fonte menor."
      "Tag h6 -Sendo utilizada na pratica nesta demonstração, em um nivel de importância tem bem menos utilização assim como a tag h5 esta apresenta menor fonte comparada a todas outras."

Conhecendo sobre a Tag p:

  • Tag p - De um jeito bem resumido e direto a tag p é utilizada para demarcar um parágrafo.


Conhecendo sobre a Tag mark:

  • Tag mark - Este elemento mark serve para dar destaque a uma parte do texto, como por exemplo um marca-texto, podendo trocar a cor da marcação lá no CSS.


Conhecendo sobre a Tag li:

  • Tag li - É muito usada para representar um item que faz parte de uma lista,
    tag da qual deve estar contido dentro de uma lista ordenada (ol), ou de uma lista desordenada (ul).


Conhecendo sobre a Tag strong:

  • Tag strong - Esta tag muito utilizada para dar um significado, uma força naquela parte especifica onde for usada, diferente da tag b que somente marca como negrito o texto, para ter uma boa semântica usar a tag strong é melhor.
